# 📊 Dashboard Comercial (Performance de Vendas)

## 🎯 Objetivo

Este projeto tem como objetivo analisar a performance de vendas de uma empresa fictícia utilizando o Power BI, explorando diferentes perspectivas para gerar insights estratégicos que auxiliem na tomada de decisão.

## 📌 Sobre o Projeto

Foram utilizados dados fictícios para simular cenários reais de negócio, permitindo a construção de dashboards interativos e análises relevantes.

## 📊 Estrutura do Dashboard

O projeto foi dividido em múltiplas visões analíticas:

### 📑 Índice (Menu Interativo)

* Navegação dinâmica entre as páginas do dashboard
* Uso de botões e bookmarks

### 🧠 Narrativa Inteligente

* Geração automática de insights com base nos dados
* Identificação de padrões e tendências

### 📈 Principais Influenciadores de Vendas

* Análise dos fatores que impactam o desempenho de vendas
* Identificação de variáveis-chave no resultado comercial

### 📊 Faixas de Vendas por Categoria e Ponto de Venda

* Segmentação de vendas por categorias de produtos
* Comparação entre diferentes pontos de venda

### 🌍 Performance dos Vendedores por Região

* Avaliação do desempenho comercial por localização
* Visualização em mapa geográfico

📌 **Regra de Negócio Aplicada:**
Foi definido um **parâmetro de filtro para destacar apenas vendedores com vendas superiores a 30.000**.
Com isso, o mapa evidencia somente os profissionais que atingiram alta performance, facilitando a identificação de regiões estratégicas e vendedores de destaque.

## 📈 Metodologia

Durante o desenvolvimento foram aplicadas as seguintes práticas:

* Uso de recursos avançados do Power BI:

## 📊 Uso de Visualizações Estratégicas

O projeto utiliza diferentes tipos de gráficos para ampliar a análise e gerar novas visões de negócio.

### 📉 Gráfico de Faixas (Band Chart)

O gráfico de faixas foi utilizado para analisar a distribuição das vendas dentro de intervalos específicos.

📌 **Por que usar esse gráfico?**

* Permite visualizar variações de desempenho entre grupos
* Ajuda a identificar padrões e dispersões nos dados
* Facilita a comparação entre categorias ou regiões
* Destaca faixas de maior concentração de vendas

📊 **Aplicação no contexto comercial:**
Esse tipo de visual é especialmente útil para:

* Entender a consistência das vendas
* Identificar outliers (valores fora do padrão)
* Avaliar concentração de faturamento em determinadas faixas

## 💡 Principais Insights (Exemplo)

* Identificação dos principais fatores que impulsionam vendas
* Diferença de performance entre regiões
* Categorias de produtos com maior relevância
* Destaque para vendedores de alta performance (acima de 30k)
* Oportunidades de melhoria no desempenho comercial
